선택: A. 도형 + 다리 모션의 미니멀 그래픽

## 근거

PRD의 목표에 비추어 독립적으로 판단한다.

- 에셋 제작 비용: 픽셀아트 스프라이트 시트는 프레임별 아트워크 제작·관리
  비용이 발생한다. 코드로 그리는 도형은 별도 아트워크 없이 구현되어 비용이
  없다.
- 외부 파일 없는 구성: PRD는 외부 오디오/이미지 파일 없는 구성을 지향한다.
  스프라이트 시트는 외부 이미지 에셋을 요구하지만, 도형 기반은 코드만으로
  완결되어 정적 호스팅 단순성을 유지한다.
- 기능 영향: 시각 표현만의 문제로 기능 요구사항에 영향이 없다. 따라서
  비용·구성 단순성이 결정 기준이 된다.

외부 에셋 의존을 배제하고 제작 비용을 없애는 A를 선택한다.
