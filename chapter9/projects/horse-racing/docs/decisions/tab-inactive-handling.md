# 결정 주제: 탭 비활성화 중 경주 처리

경주 진행 중 브라우저 탭이 비활성화되어 `requestAnimationFrame`이 중단될 때의 처리 방식을 결정한다.

## 후보

- A. 복귀 시 경과 시간만큼 시뮬레이션을 따라잡기(catch-up)
- B. 탭 비활성화 시 경주를 자동 일시정지, 복귀 시 재개

## 판단 기준

- 구현 복잡도
- 경주 결과의 신뢰성(프레임레이트·경과 시간에 따른 왜곡 방지)
- delta-time 기반 시뮬레이션과의 정합성

합의 불발. 후보를 처음부터 다시 나열하고 장단점을 비교한 뒤 결정하라.

합의: B (시도 2회)
