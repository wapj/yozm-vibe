# Odin 1.0, ‘Odin 2027’로 2027년 1월 출시 예정

- 원문: [https://news.hada.io/topic?id=31220](https://news.hada.io/topic?id=31220)
- 발행일: 2026-07-08
- 피드: GeekNews - 개발/기술/스타트업 뉴스 서비스

## 요약

Odin 프로그래밍 언어가 2016년 첫 커밋 이후 10년 만에 첫 정식 릴리스를 준비하고 있으며, 버전 이름은 날짜 기반 명명 방식을 채택한 "Odin 2027"입니다. 정식 릴리스의 타협 불가 조건은 컴파일러가 준수하는 완전한 언어 명세이며, 인라인 어셈블리, HTTP 패키지와 TLS 지원, Objective-C 코어 라이브러리, iOS·Android 지원 등이 추가될 예정입니다. 일정은 2026년 크리스마스에 Release Candidate, 2027년 1월에 정식 릴리스를 목표로 하며, 그전까지는 신규 기능 추가보다 안정화와 버그 수정에 집중합니다. 커뮤니티에서는 실용성과 신중한 설계의 균형, 1.0 릴리스가 기업의 프로덕션 도입 판단에 주는 신호 효과를 긍정적으로 평가하고 있습니다.

## 핵심 포인트

- **정식 릴리스 명칭**: 일반적인 "1.0" 대신 날짜 기반의 "Odin 2027"을 사용하여 버전의 최신성을 한눈에 파악할 수 있게 함. 기존 nightly·monthly 릴리스 체계는 유지
- **필수 조건**: 컴파일러가 따르는 완전한 언어 명세 확보가 타협 불가 기준
- **예정 기능**: 인라인 어셈블리, HTTP 패키지, TLS 지원(네이티브 TLS는 릴리스 이후), Objective-C 코어 라이브러리, iOS·Android 지원, 이미지 패키지 개선, zip 읽기/쓰기 패키지
- **일정**: 2026년 크리스마스 Release Candidate → 2027년 1월 정식 릴리스. 그전까지는 대형 신규 기능 대신 안정성, 코어 라이브러리 강화, 명세 준수, 버그 리포트 반영에 집중
- **성장 배경**: 취미 프로젝트로 시작해 JangaFX 상용 소프트웨어, 여러 회사의 주력 언어, Steam 출시 게임에 사용되는 수준으로 성장
- **커뮤니티 반응**: 실용성과 설계 완성도의 균형에 대한 호평이 많으며, 1.0 릴리스가 Jai, C3, Zig, Hare 등 다른 C 계열 신생 언어들의 기준도 높일 것이라는 의견이 있음. Odin을 "변장한 Pascal"로 보는 문법 관련 토론도 있음
