# ReactOS “오픈소스 Windows” 프로젝트, Half-Life 2 실행 가능해짐

- 원문: [https://news.hada.io/topic?id=31217](https://news.hada.io/topic?id=31217)
- 발행일: 2026-07-08
- 피드: GeekNews - 개발/기술/스타트업 뉴스 서비스

## 요약

Windows 바이너리 호환을 목표로 하는 오픈소스 운영체제 ReactOS에서 Half-Life 2가 실행되는 데 성공했습니다. 한 달 전 Half-Life 실행 데모가 공개된 지 30일도 지나지 않아 이룬 성과로, ReactOS nightly build에서 게임이 인게임 상태로 정상 동작하는 모습이 YouTube 영상으로 공개되었습니다. 테스트에는 GeForce GTX 960과 NVIDIA 368.61 레거시 드라이버, Creative Sound Blaster Audigy Windows 드라이버가 사용되었습니다. 아울러 ReactOS는 첫 Windows NT6 시스템 호출을 도입하여 Windows Vista 이후 버전과의 호환성을 향한 초기 단계에 들어섰습니다.

## 핵심 포인트

- **Half-Life 2 실행 성공**: Half-Life 데모 공개 후 30일이 지나지 않아 더 최신 게임인 Half-Life 2까지 ReactOS nightly build에서 인게임 상태로 구동됨
- **테스트 환경**: GeForce GTX 960 + NVIDIA 368.61 레거시 Windows 드라이버 + Creative Sound Blaster Audigy Windows 드라이버 조합으로 확인
- **이전 사례와의 비교**: 앞선 Half-Life 실행은 Intel Sandy Bridge 세대 데스크톱과 GeForce 8400GS라는 더 낮은 사양에서 진행됨
- **NT6 호환성 착수**: 첫 Windows NT6 시스템 호출을 도입하여 Windows Vista 이후 버전 호환성 개발의 초기 단계를 밟는 중
- **커뮤니티 반응**: 개발 속도에 대한 놀라움과 함께, 관련 YouTube 채널의 다른 게임 실행 사례(Lego Star Wars, Unreal Tournament 등)와 러시아 후원 여부에 대한 논의가 이어짐
