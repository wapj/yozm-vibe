# Chat Control 1.0과 2.0 설명

- 원문: [https://news.hada.io/topic?id=31231](https://news.hada.io/topic?id=31231)
- 발행일: 2026-07-08
- 피드: GeekNews - 개발/기술/스타트업 뉴스 서비스

## 요약

EU의 "Chat Control"은 단일 법안이 아니라 두 개의 별도 규제 흐름으로 구성됩니다. Chat Control 1.0(Regulation (EU) 2021/1232)은 제공자가 아동 성적 학대 자료 탐지를 위해 비공개 메시지를 자발적으로 스캔할 수 있게 한 임시 예외 규정으로, Parliament의 연장 거부로 2026년 4월 4일 만료되었습니다. 그러나 Council은 동일한 내용의 형식상 "새" 법안으로 신속 부활을 추진하고 있으며, 이를 저지하려면 7월 9일 표결에서 전체 MEP의 절대다수인 361명의 반대가 필요합니다. Chat Control 2.0(CSA Regulation)은 플랫폼에 탐지·신고 의무를 부과하는 영구 규제안으로, 5차례의 3자 협상에도 의심 없는 시민 메시지 스캔과 종단간 암호화 포함 여부를 두고 합의에 이르지 못했습니다. 결과적으로 임시 법은 우회 절차로 부활 중이고 영구 법은 협상 테이블에 남아 있는 상태입니다.

## 핵심 포인트

- **두 규제의 구분**: Chat Control 1.0은 자발적 스캔을 허용하는 임시 예외(이미 만료), 2.0은 탐지·신고를 의무화하려는 영구 규제안(협상 중)으로, 혼동되지만 별개의 흐름입니다.
- **1.0의 실제 적용 범위**: 스캔은 주로 암호화되지 않은 미국 서비스(Gmail, Facebook/Instagram Messenger, Skype 등)에서 이뤄졌고, 종단간 암호화 통신은 스캔된 적이 없습니다.
- **부활 절차의 문제**: 만료된 규정은 연장할 수 없으므로 Council은 동일 내용의 "새" 법을 신속 절차로 제안했고, Parliament는 책임 Committee를 건너뛰는 긴급 절차를 331–303으로 승인했습니다. 저지 기준(361명)에 미달하면 법은 자동 채택으로 간주됩니다.
- **2.0의 미해결 쟁점**: 의심받지 않는 시민의 메시지 스캔과 종단간 암호화 메시지 스캔이 Parliament와 Council 간 경계선으로 남아 있으며, Council Legal Service도 "자발적" 스캔 제안이 EU Charter 제7조와 양립하기 어렵다고 판단했습니다.
- **기업 동향**: 법적 근거 만료에도 Google, Meta, Microsoft, Snap은 비공개 메시지 스캔을 계속하겠다고 밝혔습니다.
- **커뮤니티 반응**: Hacker News에서는 기저율 오류로 인한 대량 오탐 가능성, 범죄 예방 효과에 대한 데이터 부재, 감시 확대의 선례 우려("좋은 목적을 내세운 광범위한 권한 요구") 등이 주요 비판으로 제기되었습니다.
