# Astryx - Meta가 공개한 오픈소스 디자인 시스템

- 원문: [https://news.hada.io/topic?id=31190](https://news.hada.io/topic?id=31190)
- 발행일: 2026-07-07
- 피드: GeekNews - 개발/기술/스타트업 뉴스 서비스

## 요약

Meta가 사내에서 8년간 사용해 온 디자인 시스템 Astryx를 MIT 라이선스 오픈소스로 공개했습니다. React와 StyleX 기반으로 160개 이상의 접근성 지원 컴포넌트, 다크모드, 테마, 템플릿, CLI를 하나의 통합 시스템으로 제공하며, Meta 내부에서 13,000개 이상의 앱을 구동해 온 검증된 시스템입니다. 특히 CLI와 MCP를 통해 에이전트용 문서를 자동 설정할 수 있어 AI 코딩 도구와의 결합을 전면에 내세운 점이 특징입니다. 컴포넌트 내부 구조를 개방해 어느 수준에서든 조합할 수 있고, StyleX에 종속되지 않아 Tailwind·CSS Modules 등으로 자유롭게 오버라이드할 수 있습니다.

## 핵심 포인트

- **규모와 검증**: Meta 내부에서 8년간 성장한 최대 규모 디자인 시스템으로, 13,000개 이상의 앱에서 사용되었습니다.
- **구성 요소**: 접근성·브랜드 테마를 지원하는 160개 이상의 React 컴포넌트, 다크모드, 프로덕션 레디 템플릿, CLI를 통합 제공합니다.
- **AI 에이전트 대응**: `npx astryx init` 실행으로 에이전트용 문서를 자동 설정하고, CLI 또는 MCP로 스캐폴딩·템플릿 탐색·테마 생성이 가능합니다.
- **개방된 내부 구조(Open internals)**: 컴포넌트를 어느 수준에서든 조합할 수 있고, `swizzle` 기능으로 컴포넌트 소스 전체를 프로젝트로 추출해 직접 소유할 수 있습니다.
- **스타일링 락인 없음**: 내부는 StyleX로 작성되지만 사용자에게 드러나지 않으며, `className`으로 Tailwind/CSS Modules/일반 CSS 오버라이드가 가능합니다.
- **디자인 토큰**: 색상·간격·radius·타이포그래피 등 모든 토큰을 CSS 커스텀 속성으로 제공하며 다중 테마를 지원합니다.
- **3계층 아키텍처**: Foundations(시각 기초) → Components(150개 이상 TypeScript 지원 컴포넌트) → Patterns(테이블 페이지, 폼 위저드 등 검증된 설계 솔루션)로 구성됩니다.
- **설계 철학**: 강제 대신 가이드를 우선하는 원칙(Guidance over enforcement)으로, 전달된 값을 그대로 렌더링하고 디자인 의견은 문서와 예제로 제시합니다.
- **라이선스**: MIT 라이선스로 공개되었습니다.
