# Kokoro로 로컬 CPU에서 고품질 TTS 실행하기

- 원문: [https://news.hada.io/topic?id=31239](https://news.hada.io/topic?id=31239)
- 발행일: 2026-07-08
- 피드: GeekNews - 개발/기술/스타트업 뉴스 서비스

## 요약

Kokoro는 82M 파라미터의 경량 TTS 모델로, 전용 GPU 없이 CPU만으로도 고품질 음성 합성이 가능합니다. 영어를 중심으로 중국어, 힌디어 등 여러 언어와 약 50개 음성을 지원하며, 가장 간편한 실행 방법은 음성 모델이 포함된 Kokoro-FastAPI 컨테이너(약 5GB)를 띄우는 것입니다. OpenAI speech API와 호환되는 인터페이스를 제공하므로 기존 음성 API 기반 프로그램을 로컬 TTS로 쉽게 전환할 수 있습니다. 짧은 문단 합성이 12년 전 CPU(i7-4770K)에서도 4.7초, 최신 Ryzen 7에서는 1.5초에 완료되어, GPU는 LLM 추론에 할당하고 CPU로 음성을 생성해 LLM 응답을 듣는 활용이 현실적입니다.

## 핵심 포인트

- **모델 특성**: 82M 파라미터의 소형 모델이지만 현실적인 음질을 제공하며, 영어에 가장 최적화되어 있고 약 50개 음성을 지원합니다.
- **간편한 배포**: `podman run -p 8880:8880 ghcr.io/remsky/kokoro-fastapi-cpu` 한 줄로 실행 가능하며, 웹 UI(`localhost:8880/web`)와 OpenAI 호환 API를 함께 제공합니다.
- **성능**: 짧은 문단 기준 합성 시간은 Intel Core i7-4770K 4.7초, Apple M2 Pro 4.5초, AMD Ryzen 7 8745HS 1.5초로, 구형 CPU에서도 실용적입니다.
- **대안**: Speaches는 음성 가중치를 API로 별도 다운로드하는 방식이며, Whisper 기반 STT까지 지원해 TTS와 STT를 한 곳에서 처리할 수 있습니다.
- **커뮤니티 평가**: 접근성 제품, 전자책 낭독, 웹페이지 읽기 확장 등 실사용 사례가 많으나, 한두 단어만 합성하면 발음이 불안정해지는 약점이 있어 긴 문장에 넣은 뒤 타임스탬프로 잘라내는 우회법이 쓰입니다. IPA 발음 가이드를 수동 지정할 수 있다는 점은 장점으로 꼽힙니다.
