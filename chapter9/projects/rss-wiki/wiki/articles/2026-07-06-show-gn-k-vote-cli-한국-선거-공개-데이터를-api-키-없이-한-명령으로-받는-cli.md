## 요약

k-vote-cli는 한국 선거 공개 데이터(개표결과·여론조사·투표율)를 API 키 없이 명령 한 번으로 내려받을 수 있게 만든 오픈소스 CLI입니다. 기존에는 선거통계시스템의 자동 접근 차단, EUC-KR 인코딩, 선거마다 다른 엑셀 양식, 인증키 발급 절차 등으로 접근성이 낮았는데, 이를 선관위 공식 배포 채널(data.go.kr)만 사용해 해결하고 robots.txt로 차단된 경로는 우회하지 않습니다. 원자료를 그대로 보존하고 정의가 명시된 표준 계산값(투표율·득표율·유효표)만 더한다는 원칙을 따르며, 누가 실행하든 동일한 데이터를 얻습니다. Go 단일 바이너리로 macOS/Linux/Windows를 지원하고 MIT 라이선스로 배포됩니다.

## 핵심 포인트

- **개표결과 정규화**: `kvote nec corpus --normalize`로 역대 대선(16·17·21대)·총선·지방선거 개표결과를 투표구 단위 JSONL로 변환. EUC-KR 디코딩, 동명 투표구 중복, 본투표/사전투표 분리를 자동 처리
- **여론조사 수집**: `kvote nesdc sync / bulk`로 여론조사 전수 수집. 2023년 10월 이후 정당지지율 1,400건 이상의 시계열과 표본 구성 교차표 제공
- **투표율 분석**: `kvote nec turnout-analysis`로 개표결과에 없는 성별·연령대별 투표율 축을 정규화
- **에이전트 연동**: 로컬 SQLite 적재와 read-only SQL 질의, MCP 서버 내장으로 Claude Code 같은 AI 에이전트가 탐색부터 SQL 질의까지 직접 수행 가능
- **설계 원칙**: 원자료 보존, 해석·판단 배제, 재현 가능성 보장. 인증키가 필요한 OpenAPI는 `kvote api login/apply`로 활용신청 자동화
- **제약 사항**: 일부 기능은 포털 HTML 구조에 의존하므로 구조 변경 시 `kvote doctor`로 점검 필요
