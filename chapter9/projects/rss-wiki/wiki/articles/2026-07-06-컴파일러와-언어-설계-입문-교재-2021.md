# 컴파일러와 언어 설계 입문 교재 (2021)

- 원문: [https://news.hada.io/topic?id=31164](https://news.hada.io/topic?id=31164)
- 발행일: 2026-07-06
- 피드: GeekNews - 개발/기술/스타트업 뉴스 서비스

## 요약문

Douglas Thain이 University of Notre Dame의 CSE 40243 컴파일러 수업을 위해 저술한 무료 온라인 교재 『Introduction to Compilers and Language Design』(2판, 2021년 수정)을 소개하는 글입니다. 한 학기 분량으로 컴파일러 제작 전 과정을 다루며, 최종 목표는 C와 유사한 언어를 X86 또는 ARM 어셈블리로 번역하는 간단한 컴파일러를 직접 구현하는 것입니다. 전체 및 장별 PDF와 GitHub 실습 자료가 제공되고, 개인·학술 용도로는 자유롭게 사용할 수 있으나 상업적 인쇄·배포는 금지됩니다. Hacker News 토론에서는 실제 수강 경험자의 호평과 함께, Dragon Book 대비 입문서로서의 위치, "언어 설계"라는 제목에 비해 실제로는 컴파일러 입문에 가깝다는 지적 등이 오갔습니다.

## 핵심 포인트

- **교재 개요**: Notre Dame CSE 40243 수업 기반의 무료 온라인 교재로, 전체 PDF와 장별 PDF를 내려받을 수 있으며 하드커버·페이퍼백 실물 책도 구매 가능합니다.
- **학습 목표**: C와 유사한 언어(B-Minor)를 입력받아 X86 또는 ARM 어셈블리로 변환하는 단순 컴파일러를 한 학기 동안 구현하는 것입니다.
- **대상 독자**: C 프로그래밍 경험이 있고 자료구조와 컴퓨터 구조를 이수한 학부생 수준에 가장 적합합니다.
- **부속 자료**: compilerbook-examples GitHub 저장소에 스캐너·파서 예제, 프로젝트 시작 코드, 단계별(스캐너·파서·타입체커·코드 생성기) 테스트 케이스가 제공됩니다.
- **사용 조건**: PDF는 개인·학술 용도로만 사용 가능하며, 상업적 인쇄·배포는 금지됩니다. 로컬 복사본 배포 대신 compilerbook.org 안내가 권장되고, 오류 제보는 dthain@nd.edu로 받습니다.
- **커뮤니티 반응**: 실제 수강생의 강력 추천, Dragon Book은 대학원 수준에 가깝고 입문서로는 Tiger Book이 낫다는 의견, 언어 설계보다는 컴파일러 입문에 초점이 맞춰져 있다는 평가 등이 있었습니다.
