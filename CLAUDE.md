# 한국어 응답 스타일 가이드

이 레포에서 한국어로 응답할 때는 기술 문서 수준의 문체를 유지한다.

## 금지

- 구어체·속어·개발자 슬랭: 박힘, 꽂히다, 터지다, 까이다, 갈긴다, 어그러지다, 까먹다, 멈춰버리다, 잘 안 먹힌다
- 의성어/의태어: 휙, 쓱, 후딱, 뚝딱
- 이모지
- "~인 듯", "~같음", "~인 거 같아요" 같은 추측성 단정
- 영어 감탄사/축약: "OK", "yes", "음" 등은 자제 (필요하면 "네", "맞습니다"로)

## 권장

- 종결어미는 "~합니다", "~됩니다" 등 격식체
- 명확하고 단정적인 서술. 추측이 필요하면 "~로 보입니다", "~일 가능성이 큽니다"로
- 코드/명령어/파일명은 백틱 또는 마크다운 링크

## 대체어 매핑

| 캐주얼 | 권장 |
|---|---|
| 박혀있다 / 박힘 | 고정되어 있다 / 하드코딩되어 있다 |
| 꽂히다 | 적용되다 / 주입되다 |
| 터지다 (에러) | 실패하다 / 예외가 발생하다 |
| 갈긴다 / 깐다 | 호출한다 / 실행한다 |
| 어그러지다 | 일관성이 깨지다 / 어긋나다 |
| 까먹다 | 누락하다 / 빠뜨리다 |
| 멈춰버리다 | 중단되다 |
| 잘 안 먹힌다 | 기대대로 동작하지 않는다 |
| 어쩌다 | 우연히 / 간헐적으로 |
| 그냥 | (대부분 삭제하거나 "단순히"로) |
