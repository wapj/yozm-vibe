# quote-cli

무작위 명언을 출력하는 간단한 커맨드라인 도구입니다. 인수 없이 실행하면 무작위 명언 한 건을, `--list` 옵션을 주면 전체 명언 목록을 출력합니다. 명언 데이터는 `quotes.json`에 분리되어 있어 코드 수정 없이 항목을 추가하거나 변경할 수 있습니다.

## 요구 사항

- Python 3.8 이상 (표준 라이브러리만 사용하므로 별도 런타임 의존성 없음)
- 테스트 실행 시 [pytest](https://docs.pytest.org/)

## 설치

저장소를 클론한 뒤 해당 디렉터리로 이동합니다.

```bash
git clone <repository-url>
cd quote-cli
```

테스트를 실행하려면 pytest를 설치합니다.

```bash
pip install pytest
```

## 사용법

### 무작위 명언 출력

인수 없이 실행하면 명언 한 건을 무작위로 출력합니다.

```bash
python quote.py
```

```
"오늘 할 수 있는 일에 전력을 다하라. 그러면 내일은 한 걸음 더 나아가 있을 것이다."
  - 아이작 뉴턴
```

### 전체 목록 출력

`--list` 옵션을 주면 모든 명언을 번호와 함께 출력합니다.

```bash
python quote.py --list
```

```
1. "성공은 최종이 아니고 실패는 치명적이지 않다. 중요한 것은 계속하려는 용기다."
  - 윈스턴 처칠
2. "오늘 할 수 있는 일에 전력을 다하라. 그러면 내일은 한 걸음 더 나아가 있을 것이다."
  - 아이작 뉴턴
...
```

> 환경에 따라 `python` 대신 `python3`를 사용해야 할 수 있습니다.

## 명언 데이터

명언은 `quotes.json`에 JSON 배열로 저장됩니다. 각 항목은 `text`와 `author` 필드로 구성됩니다.

```json
[
  {
    "text": "삶이 있는 한 희망은 있다.",
    "author": "키케로"
  }
]
```

항목을 추가·수정한 뒤 별도의 코드 변경 없이 바로 반영됩니다.

## 테스트

CLI의 동작을 검증하는 테스트는 `test_quote.py`에 있습니다.

```bash
python -m pytest test_quote.py -v
```

테스트는 다음을 확인합니다.

- 인수 없이 실행할 때 명언 한 건이 출력되는지
- 출력된 명언이 `quotes.json`의 실제 항목인지
- `--list`가 모든 항목을 번호와 함께 출력하는지
- `--list` 출력 줄 수가 항목 수에 맞는지
